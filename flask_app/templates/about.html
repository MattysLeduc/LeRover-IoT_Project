◈ System Architecture

YVLSWITCH is a fully integrated IoT smart robot car system featuring:

  • Raspberry Pi 4: Main controller running Python 3.11
  • Ultrasonic Sensor: HC-SR04 for distance measurement (2cm - 400cm)
  • IR Line Sensors: 3-channel array for line detection and tracking
  • Motor Driver: 4-wheel DC motor control with PWM
  • LED Array: Addressable RGB LEDs via SPI
  • Pan/Tilt Servos: Camera gimbal for scanning
  • Adafruit IO: MQTT broker for real-time cloud communication
  • Neon PostgreSQL: Cloud database for historical data
  • SQLite: Local database for offline operation
  • Flask + Chart.js: Modern web dashboard

◈ Features

  • Real-time sensor data streaming via MQTT
  • Historical data visualization with interactive charts
  • Remote motor control (forward, backward, left, right, stop)
  • Autonomous line following with PD control algorithm
  • Autonomous obstacle avoidance with pan-tilt scanning
  • Offline data caching with automatic cloud sync
  • Device control (LEDs, buzzer) via web interface
  • Responsive cyberpunk-themed UI design
